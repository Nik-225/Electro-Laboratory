# Logic
- **Архітектура**: React + Vite SPA (Single Page Application).
- **Розгортання**: Для GitHub Pages потрібно вказати правильний `base` у `vite.config.ts` (відповідає назві репозиторію) та налаштувати процес збірки (build). Найкращий сучасний підхід — використання GitHub Actions для автоматичної збірки та публікації статичних файлів з теки `dist`.
